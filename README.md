COVID-19 Data Analysis Project
This project analyzes global COVID-19 data to identify patterns, trends, and key insights related to confirmed cases, deaths, and recoveries.

Objectives
Analyze COVID-19 case trends
Visualize country-wise impact
Compare confirmed, recovered, and death rates
Perform data preprocessing and cleaning
Generate meaningful insights through visualization

Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

Features
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Graphical Visualizations
Country-wise Comparisons
Trend Analysis Over Time

File Included
covid19.ipynb – Jupyter Notebook containing complete analysis
